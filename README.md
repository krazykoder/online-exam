java-persistence-tutorial
=========================

## Overview
これは、JPA 2.1のチュートリアルとして作成したプロジェクトです。依存ライブラリは次のとおりです。

* JPA 2.1
* HIbernate EntityManager 4.3.2.Final
* H2Database 1.3.175
* Spring Framework 4.0.0.RELEASE
